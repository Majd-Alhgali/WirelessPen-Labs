# WirelessPen-Labs

مختبر محلي لتوثيق وتحليل سيناريوهات اختبار اختراق Wi-Fi داخل بيئة مصرح بها.

## المحتويات

- `Codex-Answers/`: نتائج وتحليلات مكتوبة.
- `Codex-Plans/`: خطط تنفيذ وفحص محافظة.
- `Codex-Prompts/`: تعليمات وسياق العمل المستخدم أثناء التحليل.
- `Exam-Attacks/`: ملفات لاب وسيناريوهات تدريبية.
- `TaimStarlink-Analysis_Files/`: ملفات تحليل مشتقة من الالتقاطات.
- `Terminal-OutPuts/`: مخرجات أوامر توثيقية.

## ملاحظات الأمان

هذا المستودع مخصص للتعلم والعمل داخل شبكات تملكها أو لديك إذن صريح لاختبارها. ملفات الالتقاط الخام ومواد الهاش مستثناة من Git افتراضيا لأنها قد تحتوي على بيانات حساسة أو تتجاوز حدود GitHub.
