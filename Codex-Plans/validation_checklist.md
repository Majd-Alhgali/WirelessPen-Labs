# قائمة التحقق من الصحة - Validation Checklist

## قبل البدء

### 1. فحص ملفات الـ CAP
- [ ] تحقق من وجود الملف: `taim_client_3e_test-01.cap`
- [ ] تشغيل aircrack-ng:
  ```bash
  aircrack-ng taim_client_3e_test-01.cap
  ```
  - [ ] النتيجة تظهر: `WPA (1 handshake)`
  - [ ] عدد EAPOL pairs يساوي أو أكثر من 1

### 2. تحويل إلى صيغة Hashcat
- [ ] تشغيل hcxpcapngtool:
  ```bash
  hcxpcapngtool -o taim_3e.22000 taim_client_3e_test-01.cap
  ```
  - [ ] الملف الناتج موجود: `taim_3e.22000`
  - [ ] حجم الملف > 0 بايت

- [ ] التحقق من نوع Hash:
  ```bash
  hashcat --identify taim_3e.22000
  ```
  - [ ] Mode ID: `22000` (WPA-PBKDF2-PMKID+EAPOL)
  - [ ] عدد hashes: >= 1

### 3. التحقق من ملفات التكوين
- [ ] ملف القاموس موجود: `wordlist_taim_base.txt`
  - [ ] حجم > 100 بايت
  - [ ] عدد الكلمات: >= 20

- [ ] ملف القواعس موجود: `taim_rules_light.rule`
  - [ ] حجم > 500 بايت

- [ ] ملف الأقنعة موجود: `attack_masks.txt`
  - [ ] عدد الأقنعة: >= 10

### 4. التحقق من البيئة
- [ ] Hashcat مثبت:
  ```bash
  hashcat --version
  ```
  - [ ] الإصدار >= 6.2

- [ ] hcxpcapngtool مثبت:
  ```bash
  hcxpcapngtool -h
  ```

- [ ] aircrack-ng مثبت:
  ```bash
  aircrack-ng -h
  ```

## أثناء التنفيذ

### المرحلة الأولى (Dictionary + Rules)
- [ ] الأمر بدأ بنجاح (بدون أخطاء)
- [ ] hashcat يعرض معدل التقدم
- [ ] بعد الانتهاء، تحقق:
  - [ ] "Cracked" في النتيجة = نجاح ✓ توقف هنا
  - [ ] "Not cracked" = انتقل للمرحلة الثانية

### المرحلة الثانية (Hybrid)
- [ ] 2A: Dictionary + 4 digits بدأ بنجاح
  - [ ] الانتهاء بعد 15-30 دقيقة
  - [ ] النتيجة: نجاح أو فشل
  
- [ ] 2B: Dictionary + symbol + 4 digits بدأ بنجاح
  - [ ] الانتهاء بعد 20-40 دقيقة
  - [ ] النتيجة: نجاح أو فشل
  
- [ ] إذا فشل 2A و 2B: انتقل للمرحلة الثالثة

### المرحلة الثالثة (Masks)
- [ ] 3A: Taim + أرقام بدأ بنجاح
  - [ ] انتهى بعد 10-15 دقيقة
  
- [ ] 3B: Starlink + أرقام بدأ بنجاح
  - [ ] انتهى بعد 10-15 دقيقة
  
- [ ] 3C: Taim + رمز + أرقام (اختياري)
  - [ ] انتهى بعد 15-20 دقيقة

## بعد الانتهاء

### إذا وجدت كلمة السر ✓
- [ ] سجل كلمة السر (حفظ آمن بعيداً عن التقارير العامة)
- [ ] حدد أي مرحلة أسفرت عن النتيجة:
  - [ ] المرحلة الأولى (Dictionary + Rules)
  - [ ] المرحلة الثانية - 2A (Dictionary + 4 digits)
  - [ ] المرحلة الثانية - 2B (Dictionary + symbol + digits)
  - [ ] المرحلة الثالثة - 3A (Taim mask)
  - [ ] المرحلة الثالثة - 3B (Starlink mask)
  - [ ] المرحلة الثالثة - 3C (Other mask)

- [ ] حلل النمط:
  - [ ] هل تحتوي على اسم الشبكة؟
  - [ ] هل تحتوي على سنة؟
  - [ ] هل تحتوي على رقم بسيط؟
  - [ ] هل تحتوي على رمز خاص؟
  - [ ] الطول الإجمالي: __ حرف

- [ ] وثق في التقرير:
  ```
  النمط المكتشف: [صف النمط دون الكلمة الفعلية]
  المرحلة: [رقم المرحلة]
  الزمن المستغرق: __ دقيقة
  ضعف مكتشف: [وصف السبب (اسم، سنة، رقم، إلخ)]
  ```

### إذا لم تجد كلمة السر ✗
- [ ] تأكد من فشل جميع المراحل الثلاث
- [ ] تحقق من سجلات hashcat للأخطاء
- [ ] وثق في التقرير:
  ```
  النتيجة: لم يتم استرجاع PSK
  المراحل المختبرة: الأولى + الثانية (2A + 2B) + الثالثة (3A + 3B)
  الزمن المستغرق الإجمالي: __ دقيقة
  الاستنتاج: كلمة السر قوية ضد التخمين الحالي / ليست من نمط بشري بسيط
  
  الفجوة المكتشفة:
  - غياب PMF يسمح بالتقاط handshake ✓ (نجح)
  - كسر PSK عبر التخمين ✗ (فشل - كلمة قوية)
  ```

## ملاحظات مهمة

- **لا تشغل brute force كامل:** إذا وصلت لهنا وفشل كل شيء، توقف هنا.
- **توثيق الفشل مهم:** الفشل في كسر كلمة قوية = نتيجة مهنية صحيحة
- **الخصوصية:** لا تشارك كلمة السر الفعلية، فقط النمط والضعف
